What is Bootstrap?

According to www.getbootstrap.com, "Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery."

Features
  • Large library of website templates.
  • Existing CSS for many components.
  • Easy to access from CDN, or store locally.
  • Simple and informative documentation.
  • Active open source community.
